下载 AktSQL

稳定下载链接指向 GitHub 最新 Release 的安装包。当前版本重新发布时会覆盖同名资产，因此官网链接始终保持不变。

git tag v0.1.0
git push origin v0.1.0

Windows

x64 与 ARM64 都提供 EXE 和 MSI 安装包。

Windows x64EXE 安装包AktSQL-windows-x64.exeWindows x64MSI 安装包AktSQL-windows-x64.msiWindows ARM64EXE 安装包AktSQL-windows-arm64.exeWindows ARM64MSI 安装包AktSQL-windows-arm64.msi

Linux

x86_64 与 aarch64 AppImage。aarch64 面向 64 位 ARM Linux，包括树莓派 64 位系统。

Linux x86_64AppImageAktSQL-linux-x86_64.AppImageLinux aarch64AppImageAktSQL-linux-aarch64.AppImage

macOS

Apple Silicon 版本，适用于 M 系列与 ARM 架构 macOS 设备。

macOS arm64DMG 安装包AktSQL-macos-arm64.dmg

CI 产物

Release workflow 会在三个平台构建并上传：

  • Windows x64: .exe、.msi
  • Windows ARM64: .exe、.msi
  • Linux x86_64: .AppImage
  • Linux aarch64: .AppImage
  • macOS arm64: .dmg

带版本号的副本也会附加到同一个 GitHub Release，便于归档和校验。
