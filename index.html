专注数据库工作的 SQL GUI

AktSQL Database Management

AktSQL 是面向桌面端的数据库管理工具，用于连接数据库、浏览对象树、 执行 SQL、查看结果集、编辑表结构，并保持操作逻辑贴近真实数据库模型。

下载安装包阅读指南

连接优先

管理 MySQL、MariaDB、TiDB、PostgreSQL、CockroachDB、MongoDB、SQLite、SQL Server 和 Oracle 连接配置。

结构清晰

数据库、数据表、字段、索引、约束、引擎参数和元数据都以明确对象呈现。

直接操作

创建、重命名、修改、截断和删除对象通过专用表单完成，不把 GUI 操作退化成 SQL 预览。

结果可控

查询、分页、排序和表数据浏览都以数据库执行结果为准，保持行为稳定可预期。

工作台

表结构编辑

AktSQL 将字段、索引、约束和建表 SQL 分成独立标签页，结构操作直接、明确、可追溯。

使用流程

1. 连接

创建或选择连接配置，进入数据库工作台。

2. 查看

展开数据库与数据表，查看元数据、表结构和表数据。

3. 修改

通过明确的表单和结构编辑器执行变更，需要审阅时保留可复制 SQL。
